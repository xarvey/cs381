\begin{homeworkProblem}

What we can get from Figure 1 is : \\

$T(n)=8T(\frac{n}{2})+n^2$   \hspace{30mm}         Level 0\\
$8T(\frac{n}{2})=64T(\frac{n}{4})+n^2$  \hspace{30mm}  Level 1\\
$64T(\frac{n}{4})=512T(\frac{n}{8})+n^2$ \hspace{30mm} Level 2\\
$...$ \\
$T(n) = 8^{log_{2}n}T(1)+{log_{2}n}n^2 $  \hspace{30mm}   \\

So we can conclude that $T(n)=\Theta(n^3)$

From Master theorem: $T(n)=\Theta(n^{log_{2}8})=\Theta(n^3)$ since 
$a=8$ $b=2$ $f(n)=n^2$ $where \ c=2 \ c<log_{b}a $

\medskip
\medskip


\textit{Proof}:

Basis: $T(1)=1; T(1)=1^3+log_{2}1*1=1+0=1 $ \\
Induction: $Assume\ T(n)=n^3+log_{2}n*n^2\ is\ true$ \\
From the original definition:\\
$T(n+1)=8(T(\frac{n+1}{2}))+(n+1)^2$ \\
$=8[(\frac{n+1}{2})^3+log_{2}\frac{n+1}{2}*(\frac{n+1}{2})^2]+(n+1)^2$ \\
$=8(n+1)^3+2log_{2}\frac{n+1}{2}*(n+1)^2+(n+1)^2 $ \\
$....Some\ Magic ... $ \\
$=(n+1)^3+log_{2}(n+1)*(n+1)^2 $ \\

\medskip

So by the basis and induction. We conclude that $T(n) =n^3+{log_{2}n}*n^2 $


 


\end{homeworkProblem}

\begin{homeworkProblem}


1. If I can create a boolean array from [minnum,maxnum]

\problemAnswer
{
For i from first number to last number \\
 if arr[i+k]=true or arr[i-k]=true \\
    the element exists. \\
    print it out \&\& break  \\
    
If not exists, print out not exists
}   

The complexity is $O(n)$


2. If I can not create an array.

\problemAnswer
{
MergeSort the array \\
For i from first number to last number \\
 Do binary search for i+d and i-d
    
If not exists, print out not exists
}   

The complexity is $O(nlogn)$



\end{homeworkProblem}
